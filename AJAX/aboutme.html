Over mij

Mijn naam is Peter Romijn, woonachtig in Tiel en geboren op: 22 juli 1998. Ik ben student bij ROC Midden Nederland als Applicatie- en mediaontwikkeling. En werkzaam bij BeeInteractive.

Al op jonge leeftijd (4 jaar) had ik al intresse in de computer. Toen ik 4 was, was mijn moeder verbaast dat ik een spelletje kon installeren. Ze dacht dat het eigenlijk gewoon was dat iemand van 4 dat kon, totdat mijn moeder het tegen andere ouders vertelde. Kwam mijn moeder erachter dat het uniek is dat kinderen dat kunnen. En zo bleef ik mij ontwikkelen. Op mijn 6de begon ik met het branden van spelletjes op CD's, weer een grote stap als een "klein" kind om een ICT'er te worden.
Op mijn 10de begon ik met het hosten van mijn eigen MineCraft servers te hosten! En zo heb ik toen uiteindelijk SameBestServer gestart! Een hobby project om een minecraft server te maken! Onder SameBestServer heb ik vele servers gemaakt (MineZ, survival, minigames, pretpark, Survivalen met een storyline en LingeCraft).
Toen ik steeds verder groeide in mijn ICT kennis en ik ongeveer 13 jaar was had ik mijn eerste Wordpress website gemaakt. Deze website was om te testen wat ik allemaal met Wordpress kon doen. En dat was een hele hoop zonder dat ik veel kennis had kwam ik aardig uit te voeten. Zo bleef ik groeien ook op de middelbare school. En kreeg ik basis kennis van programmeren. Daar wilde ik mee verder groeien en koos ik voor de opleiding: applicatie- en mediaontwikkeling op het ROC Midden Nederland
Op school krijg ik les in de volgende talen

  • HTML
  • CSS
  • JavaScript: vanilla / jQuery
  • PHP
  • SQL

In 2016 kreeg ik de kans om bij BeeInteractive mijn eigen server in het datacentrum te laten plaatsen! Over de kans heb ik geen gras laten groeien en ik pakte de kans gelijk met beide handen aan. Daardoor heb ik nu mijn eigen server in het datacentrum. Op deze server experimenteer ik nog steeds met MineCraft, websites en vele andere mini projectjes.

SameBestDevelopment

SameBestDevelopment is mijn eigen bedrijf. Ik ben dit bedrijf begonnen om mij te ontwikkelen binnen het bedrijfsleven. Hoe ik opdrachten binnen haal, klanten tevreden houd en opdrachten oplever. Ik heb mijn focus altijd op de klant en het product wat ik ontwikkel van de klant. Tijdens een opdracht is mijn doelstelling om de verwachtingen van de opdrachtgever te overtreffen.
