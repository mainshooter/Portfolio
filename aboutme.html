Over mij

Mijn naam is Peter Romijn. Ik beheers verschillende programmeertalen; HTML, CSS, Javascript, PHP en SQL. Ik ben woonachtig in Tiel, geboren op 22 juli in 1998, student bij ROC Midden Nederland Applicatie- en mediaontwikkeling en werkzaam bij BeeInteractive.

Al op jonge leeftijd had ik al intresse in computers. Toen ik vier was, was mijn moeder verbaasd dat ik een spelletje kon installeren. Ze dacht dat het wel gewoon was dat iemand van vier jaar dat kon, totdat ze het tegen andere ouders vertelde. Toen kwam ze erachter dat het eigenlijk uniek is dat een kind dat op die leeftijd kan. Steeds verder begon ik mezelf te ontwikkelen binnen de ICT-wereld. Op mijn zesde begon ik met het branden van spelletjes op CD'. Weer een grote stap als een kind om een ICT'er te worden.
Op mijn tiende begon ik met het hosten van mijn eigen MineCraft servers! Zo heb ik toen uiteindelijk SameBestServer gestart! Een hobby project om verschillende minecraft servers te maken! Onder SameBestServer heb ik vele servers gemaakt (MineZ, survival, minigames, pretpark, Survivalen met een storyline en LingeCraft).
Toen ik steeds verder groeide in mijn ICT kennis en ongeveer dertien jaar was had ik mijn eerste Wordpress website gemaakt. Deze website was om te testen wat ik allemaal met Wordpress kon doen, dat was een hele hoop. Zonder dat ik veel kennis had, kwam ik er aardig mee uit te voeten. Op deze manier bleef ik groeien, ook op de middelbare school bij het vak ICT kwam ik erachter dat ik me wilde verdiepen in het programmeren. Daar wil ik verder in leren. Dus koos ik voor de opleiding: applicatie- en mediaontwikkeling op het ROC Midden Nederland.
Op school krijg ik les in de volgende talen;

  • HTML
  • CSS
  • JavaScript: vanilla / jQuery
  • PHP
  • SQL

In 2016 kreeg ik de kans om bij BeeInteractive mijn eigen server in het datacentrum te laten plaatsen! Over de kans heb ik geen gras laten groeien en ik pakte de kans gelijk met beide handen aan. Daardoor heb ik nu mijn eigen server in het datacentrum. Op deze server experimenteer ik nog steeds met MineCraft, websites en vele andere mini projectjes.

SameBestDevelopment

Ik heb mijn focus altijd op de klant en het product wat ik ontwikkel van de klant. Tijdens een opdracht is mijn doelstelling om de verwachtingen van de opdrachtgever te overtreffen.
